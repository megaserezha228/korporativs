# ASCII Paint - Console ASCII Art Editor

## Краткая суть проекта

Консольное приложение на C++ для создания ASCII-арта. Программа позволяет пользователю рисовать на виртуальном холсте с помощью символов, используя различные инструменты: точка, линия, прямоугольник, заливка, ластик. Поддерживается сохранение и загрузка рисунков в текстовые файлы.

---

## Инструкция по сборке и запуску

### Требования
- CMake 3.10 или выше
- Компилятор с поддержкой C++17 (GCC, Clang, MinGW)

### Сборка

```bash
# Переход в папку с проектом
cd ascii-paint

# Создание папки для сборки
mkdir build && cd build

# Конфигурация проекта
cmake ..

# Компиляция
cmake --build .

# Запуск программы
./ascii_paint.exe
