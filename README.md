# ASCII Paint - Console ASCII Art Editor

## Краткая суть проекта

Консольное приложение на C++ для создания ASCII-арта. Программа позволяет пользователю рисовать на виртуальном холсте с помощью символов, используя различные инструменты: точка, линия, прямоугольник, заливка, ластик. Поддерживается сохранение и загрузка рисунков в текстовые файлы.

---

## Инструкция по сборке и запуску

### Требования
- CMake 3.10 или выше
- Компилятор с поддержкой C++17 (GCC, Clang, MinGW)

### Сборка


# Переход в папку с проектом
cd ascii-paint

# Создание папки для сборки
mkdir build && cd build

# Конфигурация проекта
cmake ..

# Компиляция
cmake --build .

# Запуск программы
./ascii_paint.exe

# Тесты
## Запуск тестов
Предварительные требования
- Компилятор с поддержкой C++17 (GCC, MinGW)
- Файл catch.hpp в папке tests/

#### Компиляция и запуск

```bash
# Переход в корневую папку проекта
cd ascii-paint

# Компиляция тестов
g++ -std=c++17 tests/test_canvas_catch2.cpp src/Canvas.cpp -o run_tests.exe

# Запуск тестов
./run_tests.exe

### Покрытие тестами

#### Класс `Canvas`

| Метод | Тестов покрыто | Проверяемые сценарии |
|-------|----------------|----------------------|
| Конструктор `Canvas(w,h)` | 2 | Создание с пользовательским размером, заполнение пробелами |
| `getWidth()` / `getHeight()` | 2 | Возврат корректных размеров |
| `setChar(x,y,ch)` | 2 | Установка в корректную позицию, игнорирование выхода за границы |
| `getChar(x,y)` | 2 | Чтение из корректной позиции, возврат пробела при выходе за границы |
| `drawLine(x0,y0,x1,y1,ch)` | 3 | Горизонтальная, вертикальная, диагональная линии |
| `drawRectangle(x0,y0,x1,y1,ch,fill)` | 2 | Контур прямоугольника, заливка |
| `floodFill(x,y,target,new)` | 2 | Заливка связной области, защита от одинаковых символов |
| `clear(fillChar)` | 2 | Очистка пробелом, очистка произвольным символом |
| `saveToFile(filename)` | 1 | Сохранение в файл |
| `loadFromFile(filename)` | 1 | Загрузка из файла, обработка ошибок (несуществующий файл) |

**Итого:** 14 тестовых случаев, 481 проверка (`REQUIRE`)
